# E-Sports Tournament Platform

Backend para la gestión de torneos de deportes electrónicos, inclusión de equipos, generación de brackets, estadísticas, sistema de ELO y API para integración con frontend.

## Características principales
- Autenticación con JWT
- Gestión de usuarios y equipos
- Creación y administración de torneos
- Brackets automáticos y registro de resultados
- Estadísticas y clasificación por juego
